# Portfólio Pedro Ivo

Portfólio profissional desenvolvido com HTML, CSS e JavaScript, com animações suaves, layout responsivo e área preparada para foto profissional.

## Como abrir

1. Extraia a pasta.
2. Abra no Visual Studio Code.
3. Abra `index.html` no navegador.
4. Preferencialmente use a extensão Live Server.

## Como adicionar sua foto profissional

1. Escolha uma foto vertical, de preferência com boa iluminação.
2. Renomeie o arquivo para:

`foto-profissional.jpg`

3. Coloque a imagem nesta pasta:

`assets/images/foto-profissional.jpg`

A imagem será carregada automaticamente. Caso ela não exista, o site exibirá as iniciais `PI`.

## Como adicionar o currículo

Renomeie seu currículo para:

`curriculo-seunome.pdf`

Coloque em:

`assets/documents/curriculo-seunome.pdf`

## Arquivos principais

- `index.html`: estrutura e conteúdo.
- `style.css`: visual, responsividade e animações.
- `script.js`: menu, animações, navegação ativa e fallback da foto.
